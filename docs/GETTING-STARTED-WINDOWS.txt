Hi! This guide will help you set up and run the ENFOR DATA website on your Windows computer.

üìã FIRST TIME SETUP:

Option 1 - Automatic Setup (Recommended):
   1. Right-click PowerShell and "Run as Administrator"
   2. Navigate to project folder
   3. Run: .\install-windows.ps1
   4. Follow the prompts
   5. Run: .\start-enfor-data.ps1

Option 2 - Check What's Missing:
   1. Double-click: setup-windows.bat
   2. Install missing software from the links shown
   3. Setup database as instructed
   4. Double-click: start-enfor-data.bat

Option 3 - Manual Installation:
   1. Install Node.js: https://nodejs.org/
   2. Install Go: https://golang.org/dl/
   3. Install PostgreSQL: https://www.postgresql.org/download/windows/
      - Set password to: enfor_data
   4. Setup database (see below)
   5. Double-click: start-enfor-data.bat

üóÑÔ∏è DATABASE SETUP (after installing PostgreSQL):

Method 1 - Using pgAdmin (GUI):
   1. Open pgAdmin from Start Menu
   2. Connect to PostgreSQL server (password: enfor_data)
   3. Right-click "Databases" ‚Üí Create ‚Üí Database
   4. Name it: enfor_data
   5. Open Query Tool and run:
      CREATE USER backend WITH PASSWORD 'enfor_data';
      GRANT ALL PRIVILEGES ON DATABASE enfor_data TO backend;

Method 2 - Using Command Line:
   1. Open Command Prompt as Administrator
   2. Run: psql -U postgres -h localhost
   3. Enter password: enfor_data
   4. Run these commands:
      CREATE DATABASE enfor_data;
      CREATE USER backend WITH PASSWORD 'enfor_data';
      GRANT ALL PRIVILEGES ON DATABASE enfor_data TO backend;
      \q

üîß TROUBLESHOOTING:

Problem: "PowerShell execution policy error"
Solution: Run this in PowerShell as Administrator:
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Problem: "Port already in use"
Solution: 
   - Close other applications
   - Or restart your computer
   - Or run: .\stop-enfor-data.ps1

Problem: "Database connection failed"
Solution:
   - Make sure PostgreSQL service is running
   - Press Win+R, type "services.msc"
   - Find "postgresql" service and start it
   - Check database setup steps above

Problem: "Node/Go not found after installation"
Solution:
   - Close and reopen Command Prompt/PowerShell
   - Restart your computer
   - Reinstall with "Add to PATH" option checked
